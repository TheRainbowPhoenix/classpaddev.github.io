GET STARTED

First of all, Welcome! We look forward to you joining the community.
Before you can run programs on the Classpad, install the Hollyhock-3 launcher.

1. Press Win + X and choose Windows PowerShell
2. Copy and paste the command below, then press Enter
irm https://get.classpad.dev/ | iex

Not on Windows or having issues? Join our Discord server for help!

1

Copy run.bin an .hh3 to your calculator

Copy the run.bin file to the root of your calculator's storage. Also copy any hh3 games and python apps you want to run.

2

Launch Hollyhock

Go to the Main menu, select "System", open top left "System" menu, then "Hollyhock Launcher"

3

Run your App

In the launcher, pick the app you want to run from the list and click "Run".

FEATURES

Hollyhock-3 is a feature-rich launcher. Here are a few features.

Premade Apps

There are already some programs/games that you can just use without programming knowledge.

Experimental

You can write low level code, such as assembly if you wanted, giving you access to all the hardware in the Classpad.

Utilities

There a some utilities to take a low level look at the Classpad, such as a hex editor of the loaded memory - in real time.

Documented

The launcher comes with example programs and documentation to get you started.

Fast

Programs are compiled to either binary or .hhk which allows for greater performance compared to programs in CASIO basic.

Easy to Use

Once installed, the launcher is intuitive and easy to navigate.
